# Human–AI Ledger GitHub Pages

This repository hosts the **organization GitHub Pages site** for Human–AI Ledger.

It currently lists:
- [HAIL](https://github.com/human-ai-ledger/HAIL) — normative HAIL v1.0 schema (mirrors the Zenodo record).
- [HAIL Tool](https://github.com/human-ai-ledger/hail-tool) — Python CLI/API for session capture, validation, and ledger export.
